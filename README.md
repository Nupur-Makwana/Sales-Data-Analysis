#  Sales Data Analysis & Dashboard

##  Overview

This project performs end-to-end sales data analysis using **Python (Pandas, NumPy, Matplotlib)** and builds an interactive **Power BI dashboard** to uncover key business insights.

---

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Power BI

---

##  Data Processing (Python)

* Removed missing values
* Converted data types (date, numeric)
* Created new features:

  * Month
  * Hour
  * City
* Generated **Clean  Data**

---

##  Analysis Questions

The following business questions were solved:

1. What was the best month for sales?
2. Which city had the highest sales?
3. What time should ads be displayed for maximum sales?
4. Which products sold the most?
5. What products are often sold together?

---

##  Dashboard Features (Power BI)

*  Total Sales KPI
*  Total Orders
*  Average Order Value
*  Monthly Sales Trend
*  Top 5 Products
*  Sales by City
*  Sales by Hour

---

##  Filters (Slicers)

* City
* Product
